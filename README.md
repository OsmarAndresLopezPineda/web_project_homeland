# Proyecto 5: Diseño web adaptativo : web_project_homeland

La página web del Web Project Homeland es el quinto proyecto en el programa de Desarrollo web de TripleTen. Fue creado utilizando HTML y CSS, con base en un brief de diseño creado en Figma.
En este proyecto se utiliza diseño web adaptatrrivo para smartphone, tablet y escritorion, ademas de la metodologia y estructuramiento BEM, se importan fuentes externas predeterminadas con @font-face y se utilizan fuentes responsivas, ademas se utiliza la funcion calc(), para calcular un tamaño adaptativo en las imagenes y se adapten al brieft.

# Proyecto 6: Trabajar con diseños : web_project_homeland

En esta segunta parte del proyecto implementamos varias finciones, principalmente la de Grid Loyoud para complementar la parte del diseño web adaptativo del sprint anterior, tambien aprendimos algunas herramientas adicionales como lo son el "repeat", "fr","Grid-gap" y "Grid-areas" para posicionar de mejor manera nuestro layoud.
Ademas utilizamos elementos decorativos adicionales como los son sombras y degradados.
Sin mencionar que le dimos un pequeño repaso a Github y como crear ramas, cambiar entre ramas y fusionarlas con la rama "main".
