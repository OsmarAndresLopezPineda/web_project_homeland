# Proyecto 4: Triple Espresso

La página web del Triple Espresso es el cuarto proyecto en el programa de Desarrollo web de TripleTen. Fue creado utilizando HTML y CSS, con base en un brief de diseño.
En este proyecto se utiliza la metodologia y estructuramiento BEM, se importan fuentes externas predeterminadas, menus con psudo-clases para hacer la experiencia mas dinamica, se importa contenido embebido, y se crean formularios, con las diversas caracteristicas disponibles.

## Características del proyecto

- HTML5 semántico
- Metodologia BEM
- Estructura de archivos plana BEM
- Flexbox
- Posicionamiento CSS
- Formularios HTML
- Contenido embebido
- Implementacion de fuentes externas
- Barra de navegación
- Pseudo-clases para cambiar el estilo visual
